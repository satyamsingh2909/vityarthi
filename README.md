Python Blackjack Game
Overview
This project is a simple command-line Blackjack game where a player competes against the computer dealer. It deals cards, calculates scores with Ace adjustment, and determines the winner based on classic Blackjack rules.

Features
Random card dealing with Ace value flexibility.

User interaction to hit or stand.

Automatic dealer moves following Blackjack rules.

Game loop allowing replay without restarting the program.

ASCII art logo for visual appeal.

Technologies
Python 3

random module for card dealing

art module for ASCII art display

Installation and Running
Ensure Python 3.x is installed.

Install the art package using pip install art.

Run the script:

text
python blackjack.py
Follow on-screen prompts to play.
