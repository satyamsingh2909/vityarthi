## Problem Statement
The Blackjack Game aims to simulate a card game between the user and computer where each player draws cards to get a score closest to 21 without exceeding it.

## Scope
The project provides a command-line playable Blackjack game using Python with basic game rules and decision inputs.

## Target Users
Anyone interested in learning or playing a simple CLI-based Blackjack game.

## High-Level Features
- Random card dealing
- Score calculation
- Ace value handling
- Game comparison logic
- Replay functionality
